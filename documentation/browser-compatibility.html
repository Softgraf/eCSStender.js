Browser Compatibility

eCSStender is known to be compatible with the following browsers. If you know of another it works with, please post a message to the mailing list. To test support, visit test.ecsstender.org.

  • Android 1+
  • Chrome 1+
  • Dolphin 1+
  • Flock 1+
  • Internet Explorer 6+
  • Mozilla Firefox 1.5+
  • Mozilla SeaMonkey 2+
  • Opera 9+
  • Opera Mini 5+
  • Safari 4+
  • Safari Touch
  • xScope 1+
