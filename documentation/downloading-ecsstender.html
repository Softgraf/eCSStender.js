Downloading eCSStender

eCSStender is available via several mechanisms, but you can always download the latest release by following this link.

Get the latest from Github

eCSStender’s source is maintained at Github. You can go there to watch it, fork it, or download an older release (if that's you’re thing). You can get a copy of the repository using one of the following commands:

git clone git://github.com/easy-designs/eCSStender.js.git

or

svn checkout http://svn.github.com/easy-designs/eCSStender.js.git

The project has tagged releases, "latest" always being the current production-ready build.
