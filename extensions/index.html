Extensions

The following extensions are currently available or are in development.

  • CSS3 Backgrounds and Borders

    This extension implements the CSS3 Backgrounds and Borders module back to IE6.

  • CSS3 Color

    This extension implements the CSS3 Color module back to IE6.

  • CSS3 Grid Alignment

    This experimental extension allows you to start playing with some of the proposed features of the  CSS3 Grid Positioning module.

  • CSS3 Media Queries

    An implementation of CSS3’s Media Queries.

  • CSS3 Selectors

    This extension seeks to implement the CSS3 Selectors module back to IE6. Requires assignment of an external selector engine (e.g. Sizzle, NWMatcher, etc.).

  • CSS3 Transforms

    This extension implements the CSS3 Transforms module.

  • CSS3 Transitions

    This extension implements the CSS3 Transitions module.

  • Font Face

    This extension supports the transformation of a format-agmostic font declaration into one that follows Paul Irish’s Bulletproof @font-face syntax.
